# Reference Structure Standard

This standard is derived from a private one-page Chinese PM resume sample. Do not copy or store any personal content from that sample. Use only the generic structure and layout rules below.

## Layout pattern

- Use A4 portrait with the skill's fixed Word template margins; do not change margins to force fit.
- Use a text-only layout; avoid tables, icons, photos, sidebars, and heavy color blocks.
- Center the candidate name at the top, then center one compact contact/intention line below it.
- Use compact section headings with bold text and a restrained divider or spacing cue.
- Use 楷体 for generated Word text; body text is 9.5 pt and major section headings are 10.5 pt.
- Insert one blank line before each major section heading after the first heading. Do not insert an extra blank line before the first `教育背景` heading.
- Use bold entry headings for school/company/project/organization lines.
- Use bullet paragraphs for concrete actions and results.
- Keep blank paragraphs sparse; use exactly one visual break between major sections and none before the first section.

## Recommended section sequence

For internship/campus/junior PM resumes:
1. Header: name + intention/contact line
2. 教育背景
3. 实习经历
4. 项目经历
5. 校园经历, only when it strengthens PM evidence
6. 技能

For full-time PM resumes:
1. Header: name + intention/contact line
2. 教育背景
3. 工作经历
4. 项目经历
5. 实习经历, only if highly relevant or recent
6. 校园经历, usually omit unless very strong
7. 技能

## Entry structure

Education:
- One bold line: `学校｜专业｜学历｜时间`
- Optional one short bullet for relevant courses, GPA/rank, awards, or PM-related work.

Internship/work:
- Company line: `公司｜岗位｜时间`
- Optional project line when the company contains multiple projects.
- 2-4 bullets for strongest/relevant work.
- Prefer bullets over long paragraphs.

Project:
- Project line: `项目名称｜角色｜时间/场景`
- One short context sentence only if needed.
- 2-3 bullets focused on product method, deliverables, collaboration, and results.

Campus:
- Organization/activity line.
- 1 bullet only, unless the experience strongly proves PM abilities.

Skills:
- 2-4 short skill lines grouped by category; avoid one long semicolon-separated line.
- Group by PM method, tools, data, AI/technical, and language/certificates when relevant.

## Density target

A strong one-page junior PM resume can contain roughly:
- 5-7 section headings/entry headings.
- 12-18 bullets total.
- Most bullets should be medium length; a few important bullets may be longer.
- Avoid long prose paragraphs. Use bullets as the main information carrier.
